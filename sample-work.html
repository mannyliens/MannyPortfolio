Database Hide Show

For each MB Event, a session and miscellaneous information must be created in the DB2 database. The JSP creates queries that retrieves specific information used on the registration form.

Validation Plugin Hide Show

Each MB website is implemented with the jQuery Validation plugin and a masked input plugin. The validation plugin prevents empty input fields to be sent. Whereas, the masked plugin allowed the users to input telephone numbers in a specific formats.

jQuery UI Plugin Hide Show

Most MB sites are implemented with the jQuery UI Plugin to create a better interaction experience with the user. The datepicker widget is being used to allow the user pick their desire dates with the Plugin's GUI. In additional, options and properties are included the reduce errors and mistakes among the users.
